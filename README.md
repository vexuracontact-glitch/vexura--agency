# VEXURA Agency Website

This is a static HTML website for VEXURA Agency.

## Deployment

### Vercel / Netlify / GitHub Pages
This project is a static website and can be deployed directly by pointing the platform to the root directory of this repository.

1.  **GitHub Pages:** Push your code to a GitHub repository, go to "Settings" -> "Pages", and select "Branch: main" and "Folder: / (root)".
2.  **Vercel / Netlify:** Connect your GitHub repository. The platform will automatically detect the static files and build/deploy the site.

## Development
- HTML, CSS, JavaScript
- No build process required.
